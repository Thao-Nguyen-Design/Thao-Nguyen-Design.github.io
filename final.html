FINAL PROJECT

Interactive Game

Using Processing and JavaScript code, my final project is an interactive game. The game included 5 different levels, each level represents an age milestones: toddler, childhood, adolescence, adulthood and senior. After each level, the character and the background of the game change to indicate the passing of time.

My intention with the game is showcasing the beautiful, difficult and unpredictable progress of human life. Despite the hardships and struggles, it is inspiring to see people continue trying, growing and learning from their failures.
